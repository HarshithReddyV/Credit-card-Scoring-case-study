# Credit-card-Scoring-case-study

A credit card company wants to build a credit scoring model, to predict the bad customers. The company has data on customers involving the details about the demographic and credit bureau variables. Build a scoring model on the data provided, also perform model validation to check how well the model will perform on new data.

Required output:

Presentation format wth findings from the scoring model. Measures of model accuracy. Code used for modelling, log file output, and output of model, all in presentation format.
